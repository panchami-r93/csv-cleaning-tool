# CSV Data Cleaning Tool

Automated web app for cleaning and preparing messy CSV files.

## ✨ Features
- ✓ Remove duplicate rows
- ✓ Handle missing values (auto-fill)
- ✓ Remove extra spaces
- ✓ Standardize text formatting
- ✓ Real-time preview
- ✓ One-click download
